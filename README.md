# 📊 Simulador de Investimentos (Calculadora de Juros)

Uma aplicação interativa de terminal (CLI) desenvolvida em Java para simulação de cenários financeiros, incluindo cálculos de Juros Simples e Compostos.

## 🎯 Objetivo do Projeto
Este projeto foi construído para consolidar os fundamentos da linguagem Java (estruturas de controle, laços de repetição, leitura de dados e formatação de saída) aplicados a regras de negócio reais do setor financeiro. O sistema garante que a entrada de dados seja validada e que a saída gere um "extrato" claro e formatado para o usuário.

## 🚀 Funcionalidades
- **Menu Interativo:** Navegação contínua em loop até que o usuário decida encerrar a aplicação.
- **Cálculo de Juros Simples:** Projeção de rendimentos com base em capital, taxa e tempo. (Em desenvolvimento)
- **Cálculo de Juros Compostos:** Simulação de crescimento exponencial de investimentos. (Em breve)

## 🛠️ Tecnologias Utilizadas
- **Java:** Lógica principal e cálculos matemáticos.
- **Scanner:** Captura de dados e interação em tempo real via terminal.
- **System.out.printf:** Formatação precisa de dados financeiros para a geração de extratos.
